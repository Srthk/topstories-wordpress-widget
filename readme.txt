A simple widget plugin to display posts from a category on the sidebar with a minimalstic design (mostly for trending/top-stories posts)

== Description ==
Simply chose the category for displaying posts in the widget along with the number of posts to display. The minimalstic design of the widget, makes it look good on every theme.


== Installation ==
This section describes how to install the plugin and get it working.

1. Simply Upload/Install the widget plugin and activate
2. Go to Appearance > Widget and find this widget
3. Drag the widget to your preferred location sidebar
4. Chose the title of the widget
5. Add the number of posts to display
6. Chose the category from which to pick the posts
7. Enjoy your widget

== Frequently Asked Questions ==
= To change the font of the widget? =

By default, the font is "Oswald", which can be simply configured by changing the .css file with the widget. In the next version, it\'ll be hopefully configurable.
